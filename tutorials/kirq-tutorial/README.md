# Kirq-tutorial

This is the Kirq tutorial repository. You will find tutorials intended to users of the testbed, to ease the process of integration on our infrastructure.

## Run the code

1. To run the tutorial without showing the code :

```commandline
marimo run tutoriel.py
```

2. To run the tutorial and play with the code :

```commandline
marimo edit tutoriel.py
```

## Installation

You should have [marimo](https://marimo.io/) installed on your machine in order to run the code. 

```commandline
pip install marimo
```

All other dependencies will be installed within the virtual environment hosted by marimo.
